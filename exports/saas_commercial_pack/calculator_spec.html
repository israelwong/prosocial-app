Especificación Calculadora de Planes

Objetivo: permitir a prospectos estimar el plan ideal según usuarios, proyectos activos/mes y storage, con add-ons automáticos.

Inputs

  • Usuarios internos (1–50)
  • Proyectos activos por mes (1–200)
  • Almacenamiento requerido (GB) (5–1000)
  • Dominios personalizados (0–5)
  • Periodo (mensual/anual)

Lógica

  1. Seleccionar plan base por umbrales: Básico ≤2/≤5/≤5GB; Pro ≤5/≤20/≤25GB; Premium ≤15/≤60/≤100GB.
  2. Excesos: usuarios extra = USD 5/u/mes; proyectos extra = USD 1–2 según plan; storage extra = escalones (+50GB=USD10, +200GB=USD30).
  3. Dominios: Premium incluye 1; adicionales USD 5/mes.
  4. Aplicar descuento anual 20%.

Outputs

ConceptoMonto
Plan baseUSD X
Usuarios extraUSD X
Proyectos extraUSD X
Storage extraUSD X
Dominios extraUSD X
TotalUSD X

Mensajes: mostrar plan recomendado y motivadores (ej. “Ahorra 20% pagando anual”).
